## 🧠 About the Project

**Study Companion** is a web app built for students who want to organize their study life in one place.

I originally started this as a **college project**, but left it unfinished in between.
Now I am continuing and improving it for the **GitHub Finish-Up-A-Thon Challenge on DEV Community**.

The goal is simple:

> Turn an incomplete study planner into a complete AI-powered student productivity dashboard.

<br>

## 🎯 What the App Helps With

* Organizing subjects and topics
* Creating study tasks

## 🖥️ Run Locally

Clone the repository:

```bash
git clone https://github.com/amritkang165/study-companion.git
```

Move into the project folder:

```bash
cd study-companion
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

<br>

## 🔐 Environment Variables

Create a `.env` file in the root folder if AI features are enabled.

```env
VITE_AI_API_KEY=your_api_key_here
```

> Do not commit your real API key to GitHub.
